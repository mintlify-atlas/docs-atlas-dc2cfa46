## What is tsim?

tsim is a GPU-accelerated quantum circuit simulator designed for fast sampling from Clifford+T circuits with Pauli noise. Built on ZX-calculus stabilizer rank decomposition, tsim extends the familiar Stim API with support for non-Clifford gates including T, T_DAG, and arbitrary rotation gates (R_X, R_Y, R_Z, U3).

The simulator leverages JAX for efficient compilation and can handle hundreds of qubits, provided the circuit doesn't contain too many non-Clifford instructions.

## Quick Example

```python
import tsim

# Create a circuit with non-Clifford gates
c = tsim.Circuit("""
    RX 0
    T 0
    H 0
    CNOT 0 1
    M 0 1
    DETECTOR rec[-1] rec[-2]
""")

# Compile and sample
detector_sampler = c.compile_detector_sampler()
samples = detector_sampler.sample(shots=1000)
```

## How It Works

tsim uses **stabilizer rank decomposition** together with **ZX-calculus simplification rules** to efficiently simulate quantum circuits. The approach, based on [Sutcliffe and Kissinger (2025)](https://arxiv.org/abs/2403.06777), decomposes circuits into manageable components that can be sampled efficiently.

Just like Stim, tsim compiles circuits into measurement or detector samplers. These samplers manage a contiguous data structure optimized for efficient batch sampling on both CPU and GPU.
