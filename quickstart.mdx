This guide walks you through creating and sampling your first quantum circuit with tsim.

## Installation

First, install tsim using pip:

```bash
pip install bloqade-tsim
```

For GPU acceleration with CUDA 13:

```bash
pip install "bloqade-tsim[cuda13]"
```

See the [Installation](/installation) guide for more options.

## Your First Circuit

<Steps>
  <Step title="Import tsim">
    Start by importing the tsim library:

    ```python
    import tsim
    ```
  </Step>

  <Step title="Create a Circuit">
    Create a quantum circuit using Stim's text format. tsim supports all Stim gates plus additional non-Clifford gates:

    ```python
    circuit = tsim.Circuit("""
        H 0
        CNOT 0 1
        M 0 1
    """)
    ```

    This creates a simple Bell state preparation and measurement circuit.
  </Step>

  <Step title="Compile a Sampler">
    Compile the circuit into a measurement sampler:

    ```python
    sampler = circuit.compile_sampler()
    ```

    This preprocessing step analyzes the circuit and prepares optimized data structures for fast sampling.
  </Step>

  <Step title="Sample Measurements">
    Sample measurement outcomes from the circuit:

    ```python
    samples = sampler.sample(shots=1000)
    print(samples)
    ```

    The result is a NumPy array of shape `(1000, 2)` containing boolean measurement outcomes.
  </Step>
</Steps>

## Complete Example

Here's a complete example demonstrating tsim's key features:

```python
import tsim

# Create a circuit with non-Clifford gates and noise
circuit = tsim.Circuit("""
    H 0
    T 0
    CNOT 0 1
    DEPOLARIZE2(0.01) 0 1
    M 0 1
    DETECTOR rec[-1] rec[-2]
""")

# Sample detectors
detector_sampler = circuit.compile_detector_sampler()
samples = detector_sampler.sample(shots=1000)

print(f"Circuit has {circuit.num_qubits} qubits")
print(f"Circuit has {circuit.num_detectors} detectors")
print(f"Sampled {samples.shape[0]} shots")
```

## Using Non-Clifford Gates

tsim extends Stim with support for non-Clifford gates:

<CodeGroup>

```python T and T_DAG Gates
circuit = tsim.Circuit("""
    H 0
    T 0
    T_DAG 1
    CNOT 0 1
    M 0 1
""")
```

```python Rotation Gates
circuit = tsim.Circuit("""
    R_X(0.5) 0
    R_Y(0.25) 1
    R_Z(1.0) 2
    M 0 1 2
""")
```

```python Universal U3 Gate
circuit = tsim.Circuit("""
    U3(0.5, 0.25, 0.125) 0
    M 0
""")
```

</CodeGroup>

Rotation angles are specified as multiples of π. For example, `R_X(0.5)` applies a rotation of π/2 around the X-axis.

## Working with Detectors and Observables

tsim supports detectors and observables for quantum error correction:

```python
import tsim

# Create a repetition code circuit
circuit = tsim.Circuit("""
    # Prepare |0⟩
    R 0 1 2
    
    # Stabilizer measurements
    CNOT 0 1
    CNOT 1 2
    M 1
    DETECTOR rec[-1]
    
    # Logical observable
    M 0 2
    OBSERVABLE_INCLUDE(0) rec[-1] rec[-2]
""")

# Sample detectors and observables
sampler = circuit.compile_detector_sampler()
det_samples, obs_samples = sampler.sample(
    shots=1000,
    separate_observables=True
)

print(f"Detector samples: {det_samples.shape}")
print(f"Observable samples: {obs_samples.shape}")
```

<Warning>
**Important**: tsim detector semantics differ from Stim. tsim reports actual detector values, while Stim reports detector *flips* (XOR with a reference sample). See the [Stim Compatibility](/guides/stim-compatibility) guide for details.
</Warning>

## Performance Tips

<AccordionGroup>
  <Accordion title="Use Batch Sampling">
    Sample multiple shots at once to maximize GPU utilization:

    ```python
    # Good: Large batch size for GPU
    samples = sampler.sample(shots=10000, batch_size=2048)
    
    # Bad: Small batch size underutilizes GPU
    samples = sampler.sample(shots=10000, batch_size=10)
    ```
  </Accordion>

  <Accordion title="Minimize T-Count">
    Circuits with fewer T gates compile and run faster:

    ```python
    # Check T-count before running
    t_count = circuit.tcount()
    print(f"Circuit has {t_count} T gates")
    
    # Aim for T-count < 20 for fast compilation
    ```
  </Accordion>

  <Accordion title="Reuse Compiled Samplers">
    Compile once, sample many times:

    ```python
    # Compile once
    sampler = circuit.compile_sampler(seed=42)
    
    # Sample multiple times (fast)
    for _ in range(100):
        samples = sampler.sample(shots=1000)
    ```
  </Accordion>
</AccordionGroup>

## Inspecting Circuits

tsim provides methods to inspect circuit properties:

```python
circuit = tsim.Circuit("""
    H 0
    T 0
    CNOT 0 1
    M 0 1
    DETECTOR rec[-1]
""")

print(f"Qubits: {circuit.num_qubits}")
print(f"Measurements: {circuit.num_measurements}")
print(f"Detectors: {circuit.num_detectors}")
print(f"T-count: {circuit.tcount()}")
print(f"Instructions: {len(circuit)}")

# View circuit as text
print(circuit)
```
