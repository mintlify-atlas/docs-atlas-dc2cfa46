## Overview

The `Circuit` class is a thin wrapper around `stim.Circuit` that adds support for non-Clifford gates like T gates and arbitrary rotations. Circuits are constructed using Stim program text and support all standard Clifford operations plus:

- T and T_DAG gates (implemented via parametric rotations)
- Arbitrary rotations: R_X, R_Y, R_Z with angle parameters
- U3 gates with three rotation angles

## Constructor

```python

```

Initialize a circuit from Stim program text.

<ParamField path="stim_program_text" type="str" default="">

</ParamField>

**Example:**


## Instance Methods

### `append`

```python
append(
    name: Union[str, stim.CircuitInstruction, stim.CircuitRepeatBlock, stim.Circuit],
    targets: Union[int, stim.GateTarget, stim.PauliString, Iterable[...]] = (),
    arg: Union[float, Iterable[float], None] = None,
    *,
    tag: str = ""
) -> None
```

Append an operation into the circuit.

<ParamField path="name" type="str | stim.CircuitInstruction | stim.CircuitRepeatBlock | stim.Circuit">
  The name of the operation's gate (e.g. "H", "M", "CNOT", "T", "R_X", "U3") or a stim instruction/block.
</ParamField>

<ParamField path="targets" type="int | stim.GateTarget | stim.PauliString | Iterable" default="()">
  The objects operated on by the gate. Can be a single qubit index or an iterable of targets.
</ParamField>

<ParamField path="arg" type="float | Iterable[float] | None" default="None">
  The parens arguments for the gate. For rotation gates (R_X, R_Y, R_Z), a single angle in units of π. For U3 gates, three angles [theta, phi, lambda] in units of π.
</ParamField>

<ParamField path="tag" type="str" default="">
  A customizable string attached to the instruction.
</ParamField>

**Examples:**

```python
import tsim

circuit = tsim.Circuit()

# Append Clifford gates
circuit.append("H", 0)
circuit.append("CNOT", [0, 1])

# Append T gate
circuit.append("T", 0)

# Append rotation gates
circuit.append("R_X", 0, 0.5)  # π/2 rotation
circuit.append("R_Y", 1, 0.25)  # π/4 rotation

# Append U3 gate
circuit.append("U3", 0, [0.5, 0.25, 0.125])  # theta, phi, lambda

# Append measurements
circuit.append("M", [0, 1])
```

### `detector_error_model`

```python
detector_error_model(
    *,
    decompose_errors: bool = False,
    flatten_loops: bool = False,
    allow_gauge_detectors: bool = False,
    approximate_disjoint_errors: bool = False,
    ignore_decomposition_failures: bool = False,
    block_decomposition_from_introducing_remnant_edges: bool = False
) -> stim.DetectorErrorModel
```

Return a stim.DetectorErrorModel describing the error processes in the circuit.

<ParamField path="decompose_errors" type="bool" default="False">
  When True, attempt to decompose composite error mechanisms into simpler errors. Not supported by tsim.
</ParamField>

<ParamField path="flatten_loops" type="bool" default="False">
  When True, the output will not contain any repeat blocks.
</ParamField>

<ParamField path="allow_gauge_detectors" type="bool" default="False">
  When False, verify that detectors are deterministic under noiseless execution.
</ParamField>

<ParamField path="approximate_disjoint_errors" type="bool | float" default="False">
  When True, treat disjoint error components as independent. Can also be a probability threshold.
</ParamField>

<ParamField path="ignore_decomposition_failures" type="bool" default="False">
  When True, undecomposed errors are inserted into the output instead of causing errors.
</ParamField>

<ParamField path="block_decomposition_from_introducing_remnant_edges" type="bool" default="False">
  Requires both components to exist elsewhere before allowing decomposition.
</ParamField>

<ResponseField name="returns" type="stim.DetectorErrorModel">
  A detector error model describing how errors propagate to detectors and observables.
</ResponseField>

**Example:**

```python
import tsim

circuit = tsim.Circuit('''
    H 0
    CNOT 0 1
    X_ERROR(0.01) 0 1
    M 0 1
    DETECTOR rec[-1] rec[-2]
''')

dem = circuit.detector_error_model(approximate_disjoint_errors=True)
```

### `diagram`

```python
diagram(
    type: Literal["pyzx", "pyzx-dets", "pyzx-meas", "timeline-svg", "timeslice-svg"] = "timeline-svg",

    filter_coords: Iterable[Iterable[float] | stim.DemTarget] = ((),),
    rows: int | None = None,
    height: float | None = None,

```

Return a diagram of the circuit.

<ParamField path="type" type="str" default="timeline-svg">
  The type of diagram: "pyzx" (ZX diagram), "pyzx-dets" (ZX for detectors), "pyzx-meas" (ZX for measurements), "timeline-svg" (operations over time), or "timeslice-svg" (operations between TICKs).
</ParamField>

<ParamField path="tick" type="int | range | None" default="None">
  Required for timeslice diagrams. Specifies which TICK instruction or range to display.
</ParamField>

<ParamField path="filter_coords" type="Iterable" default="((),)">
  Filter what to include in the diagram (detector coordinates, DemTargets, etc).
</ParamField>

<ParamField path="rows" type="int | None" default="None">
  Number of rows for multi-piece diagrams.
</ParamField>

<ParamField path="height" type="float | None" default="None">
  Optional height for the rendered diagram.
</ParamField>

<ParamField path="width" type="float | None" default="None">
  Optional width for the rendered diagram.
</ParamField>

<ResponseField name="returns" type="Any">
  An object whose __str__ method returns the diagram. May define _repr_html_ for notebook display.
</ResponseField>

**Example:**

```python
import tsim

circuit = tsim.Circuit('''
    H 0
    CNOT 0 1
    T 0 1
    M 0 1
''')

# Timeline diagram
svg = circuit.diagram(type="timeline-svg")

# ZX diagram
zx = circuit.diagram(type="pyzx")
```

## Operators

### Addition (`+`, `+=`)

Concatenate circuits together.

**Example:**

```python
import tsim

c1 = tsim.Circuit("H 0")
c2 = tsim.Circuit("CNOT 0 1")

# Create new circuit
c3 = c1 + c2

# In-place addition
c1 += c2
```

### Multiplication (`*`, `*=`)

Repeat a circuit multiple times.

**Example:**

```python
import tsim

c1 = tsim.Circuit("H 0\nCNOT 0 1")

# Create new repeated circuit
c2 = c1 * 5
c3 = 3 * c1

# In-place repetition
c1 *= 10
```

### Indexing (`[]`)

Access individual instructions or slices.

**Example:**

```python
import tsim

circuit = tsim.Circuit("H 0\nCNOT 0 1\nM 0 1")

# Get single instruction
instr = circuit[0]  # H 0

# Get slice
subcircuit = circuit[0:2]  # H 0 and CNOT 0 1
```

### Equality (`==`)

Check if two circuits are exactly equal.

**Example:**

```python
import tsim

c1 = tsim.Circuit("H 0\nCNOT 0 1")
c2 = tsim.Circuit("H 0\nCNOT 0 1")

assert c1 == c2
```

### Length (`len()`)

Get the number of instructions in the circuit.

**Example:**

```python
import tsim

circuit = tsim.Circuit("H 0\nCNOT 0 1\nM 0 1")
print(len(circuit))  # 3
```
